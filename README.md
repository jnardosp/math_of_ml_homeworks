# Math of ML homeworks
A repo for all homeworks of "Matemáticas del aprendizaje de máquina" course in Universidad Nacional de Colombia (UNAL).

## Homework 1: Convex Optimization related to Hard Margin SVMs
We explore the **convex optimization problem** related to **Support Vector Machines (SVMs)**, covering the mathematical problem setup, **convexity proofs**, **Lagrange multipliers**, **KKT conditions**, and the transition from **primal to dual** optimization problems for SVM training.
